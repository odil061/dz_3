# Домашнее задание к работе 3
Условие задачи
Написать и отладить программу вычисления объема куба и
площади его боковой поверхности по заданной длине ребра.
## 1. Алгоритм и блок-схема
### Алгоритм
1. **Начало**
2. Ввод входных данных:
   - `a`- длина куба
   - `b`- ширина куба
   - `c`- висота куба
3. Считаем:
   - `kub = a * b * c`
   - `ploshad = 2 * (a * b + a * c + b * c)`
4. Выводим результаты расчетов:
   - `kub`
   - `ploshad`
5. **Конец**
